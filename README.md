# Technologies used
### Frontend(Deployed on Netlify)
- ReactJS
- Socket.io-client
- React-router-dom
- Axios
- ContextAPI
- React-toastify
- MaterialUI
- Fortawesome

### Backend(Deployed on Heroku)
- NodeJS
- ExpressJS
- Socket.io
- MongoDB
- Mongoose
- PassportJS
